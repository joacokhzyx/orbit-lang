# Photon Native Feature Coverage

This document outlines the coverage status of compiler backend features compared between the native x86-64 target and the C reference backend (Steel).

## Feature Grid

*   [x] Basic blocks and CFG structures.
*   [x] Standard calling convention (Windows x64 & System V AMD64).
*   [x] Direct relocatable object output (COFF & ELF64).
*   [ ] Structured unions and memory-aligned structures.
*   [ ] Native dynamic memory and Arena-based allocations.
*   [ ] Self-hosted binary linking (Sovereign Linker).
